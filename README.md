این پروژه شامل یک تحلیل اکتشافی جامع داده‌ها (EDA) بر روی مجموعه داده Pima Indians Diabetes است که در Kaggle موجود است. هدف اصلی این EDA، درک ساختار داده‌ها، بررسی توزیع ویژگی‌ها، شناسایی همبستگی‌ها و کشف الگوهای نهفته‌ای است که به پیش‌بینی یا درک بهتر عوامل مرتبط با دیابت کمک می‌کنند. داده‌ها شامل چندین ویژگی پزشکی و پیامد نهایی (تشخیص دیابت) برای زنان سرخپوست قبیله پیما است.
این تحلیل از مجموعه داده Pima Indians Diabetes استفاده می‌کند که شامل اطلاعات تشخیصی بیماران است.
